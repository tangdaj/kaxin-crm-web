# 基于Vue.js和Element-ui的后台管理系统

> ## 1.项目运行：
- git clone 到本地
- cd 到 vue_element_shop_manage目录下运行:

```
npm install 或者 cnpm install
```
- 运行：

```
npm run serve
```
- 浏览器打开 http://183.215.190.121:8080/
